## Run Locally

**Prerequisites:**  Node.js
1-Extract project files into a folder
   <br></br>

2- open cmd in your folder path
   <br></br>

3. Install dependencies:
   `npm install`
   <br></br>
   
4. Run the app:
   `npm run dev`
